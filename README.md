# Design Patterns in C# (.Net Core 3.0)
###### This is a console application with examples of design patterns in C# 

## Design Patterns included:

- Singleton
- Factory
- Memento
- Builder
- Prototype
- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Proxy
- Chain of Responsibility
- Command
- Iterator
- Mediator
- Observer
- State
- Strategy
- Template Method
- Visitor
